## 🚀 Sobre INGENIATEC

El semillero de investigación **INGENIATEC** surge como un espacio de formación extracurricular que promueve la **innovación tecnológica**, el **prototipado de soluciones** y la **experimentación activa** mediante herramientas como la impresión 3D, la electrónica y la robótica.

Somos un grupo comprometido con el desarrollo de soluciones innovadoras que puedan impactar positivamente en la sociedad. A través del aprendizaje colaborativo y la experimentación práctica, transformamos ideas en prototipos funcionales.

### 🎯 Misión

Fomentar la cultura de la investigación y la innovación tecnológica en estudiantes de la Corporación Universitaria Americana, proporcionando espacios de formación práctica y desarrollo de proyectos interdisciplinarios.

### 👁️ Visión

Ser reconocidos como el semillero líder en innovación tecnológica en la región Caribe, formando investigadores capaces de desarrollar soluciones tecnológicas sostenibles y de impacto social.

## 🏢 WorkinnLab

INGENIATEC está adscrito a **WorkinnLab**, el laboratorio de innovación de la Corporación Universitaria Americana. WorkinnLab es un ecosistema de innovación donde convergen ideas, tecnologías y talentos, contando con:

- 🖥️ Equipos especializados de última generación
- 🤝 Comunidad colaborativa de innovadores
- ⚡ Recursos tecnológicos avanzados
- 🏆 Respaldo institucional para proyectos

## 📋 Líneas de Trabajo

Nuestro semillero se desarrolla en cuatro líneas principales de investigación y desarrollo:

### 🧊 Impresión 3D
Prototipado rápido y fabricación aditiva para materializar ideas en objetos tangibles.
- Modelado 3D CAD
- Prototipado rápido
- Diseño de piezas personalizadas
- Fabricación aditiva FDM

### 🔌 Electrónica
Desarrollo de circuitos, sistemas embebidos y dispositivos IoT para automatización.
- Arduino & ESP32
- Sistemas IoT
- Sensores y actuadores

### 🤖 Robótica
Creación de robots autónomos y sistemas mecatrónicos inteligentes.
- Robótica móvil
- Automatización industrial
- IA embebida
- Sistemas de control

### 💻 Desarrollo de Software
Creación de aplicaciones con las últimas tecnologías y metodologías ágiles.
- Aplicaciones Web & Móviles
- Python
- desarrollo con IA
- Metodologías ágiles

## 🛠️ Sobre el Sitio Web

> *Esta sección es informativa sobre la construcción del portal web.*

El sitio web oficial de INGENIATEC fue desarrollado con un enfoque **moderno, futurista y minimalista**, inspirado en diseños de empresas tecnológicas de vanguardia como Ghostty.

### Tecnologías Utilizadas

| Tecnología | Versión | Propósito |
|------------|---------|-----------|
| **Bootstrap** | 5.3.2 | Framework CSS para diseño responsive |
| **HTML5** | 5 | Estructura semántica del contenido |
| **CSS3** | 3 | Estilos personalizados y animaciones |
| **JavaScript** | ES6+ | Interactividad y funcionalidades |
| **Bootstrap Icons** | 1.11.1 | Iconografía del sitio |
| **Google Fonts** | - | Tipografías Orbitron, Rajdhani, JetBrains Mono |

### Características del Diseño

- 🎨 **Modo oscuro único** - Optimizado para visualización nocturna
- 🖥️ **Hero terminal estilo Ghostty** - Pantalla completa con terminal interactiva
- 🤖 **Robot asistente animado** - Elemento visual protagonista con animaciones CSS
- 📱 **100% Responsive** - Adaptable a todos los dispositivos
- ⚡ **Animaciones fluidas** - Transiciones y efectos visuales modernos
- 🎯 **Navegación intuitiva** - Menú fijo con scroll suave

### Estructura del Proyecto

```
web_semilleros/
├── index.html          # Página principal
├── css/
│   └── styles.css      # Estilos personalizados (2200+ líneas)
├── js/
│   └── main.js         # Funcionalidades JavaScript
├── assets/
│   └── logotipos/      # Recursos gráficos
└── README.md           # Este archivo
```

### Secciones del Sitio

1. **Hero Terminal** - Portada inmersiva con terminal de comandos
2. **Nosotros** - Información del semillero y estadísticas
3. **Equipo** - Galería de los 13 miembros
4. **Líneas de Trabajo** - Las 4 áreas de enfoque
5. **Galería** - Espacio para contenido futuro
6. **WorkInnLab** - Información del laboratorio
7. **Contacto** - Formulario y redes sociales
